# Spec Template

Use this template when creating a new game feature.

- Title: Short descriptive name
- Summary: One-paragraph overview of the game and player experience
- Goals: What success looks like
- Scope: What is included in MVP and what is out-of-scope
- Users: Player types and personas
- Requirements:
  - Functional: game rules, turns, win conditions, UI screens
  - Non-functional: desktop layout, input devices (mouse/keyboard), accessibility
- Assets: art, sounds, fonts
- Milestones: spec review, prototype, QA, release
- Acceptance Criteria: clear pass/fail items
- Notes: references and open questions
