Skeleton for adding a new game feature.

Steps:

1. Copy this folder to `features/<game-name>/`.
2. Fill `spec.md` with product specs (PM Lead) and open a PR to `feature/<game-name>/specs`.
3. After spec approval and merge, engineering creates the implementation branch and tasks.

Files in this folder:

- `spec-template.md`: Template for writing game specs.
- `IMPLEMENTATION_GUIDE.md`: Engineering guidelines for implementing a game.
